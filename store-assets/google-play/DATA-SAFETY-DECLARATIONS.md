# Google Play Data Safety Declarations

## Data Collected
- **Personal Info**: Email address, Name (optional), User IDs (managed via Supabase).
- **Financial Info**: Purchase history (managed via RevenueCat/Stripe).
- **App Info and Performance**: Crash logs, diagnostics, and usage data.

## How Data is Used
- App functionality (Authentication, progress syncing).
- Account management and parental controls.
- Analytics and continuous curriculum improvement.

## Data Sharing
- No personal data is sold to third parties.
- Shared only with infrastructure providers (Supabase, Mixpanel, RevenueCat, Stripe) strictly for app functionality.

## Security Practices
- **Encryption**: All data is encrypted in transit using HTTPS.
- **Data Deletion**: Users can request data deletion directly within the application (Account Settings -> Delete Account) or via the support website.
