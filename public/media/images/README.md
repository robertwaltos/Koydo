# Media Image Slots

Drop finalized image assets in these paths so UI and media workflows can use them directly:

- `public/media/images/heroes/`
  - `hero-forest-01.webp`
  - `hero-ocean-01.webp`
  - `hero-space-01.webp`
  - `hero-candy-01.webp`
  - `hero-sunrise-01.webp`
  - `hero-city-01.webp`

- `public/media/images/grounds/`
  - `bg-forest-ambient-01.webp`
  - `bg-ocean-ambient-01.webp`
  - `bg-space-ambient-01.webp`
  - `bg-candy-ambient-01.webp`
  - `bg-sunrise-ambient-01.webp`
  - `bg-city-ambient-01.webp`

Recommended export sizes:

- Hero scenes: `1920x1080`
- Ambient backgrounds: `2560x1440`
- Module cards: `1024x1024`

If a named file is missing, Koydo falls back to local generated images in `public/media/generated/`.
