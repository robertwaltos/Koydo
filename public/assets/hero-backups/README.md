# Hero Backup Assets

This directory contains experimental backup hero asset sets.

- No files here are wired to runtime by default.
- Each `set-*` folder is a distinct visual direction.
- `core/` contains semantic hero slots.
- `microapps/` contains per-app hero variants.

Manifests:
- `HERO-BACKUP-GENERATION-MANIFEST.json` - OpenAI attempt status (currently blocked by billing cap).
- `HERO-BACKUP-PROCEDURAL-MANIFEST.json` - completed procedural generation output status.
