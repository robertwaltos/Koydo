## About Koydo

Koydo is a premium AI-powered educational platform delivering personalized learning across **Web**, **iOS**, and **Android**. The platform serves learners from Pre-K through adult professional development with adaptive curriculum, interactive assessments, and intelligent tutoring — all wrapped in a child-safe, COPPA-compliant experience.
